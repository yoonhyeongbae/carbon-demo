\# carbon\_demo



조직경계 기반 탄소배출 프레임워크에서 탄소배출량 산정 및 최적화를 위한 웹 기반 SaaS 소프트웨어 데모 버전


\# 주요 기능

\- 샘플 CSV 데이터 읽기
\- 기준 탄소배출량 계산
\- OR-Tools 기반 최적화
\- 기준안과 최적안 비교 표시


\# 해야할 일

\# 사용자 실행 방법 웹 앞부분에 내용 추가 (추후 업데이트 될 예정)
1. 배포된 streamlit 주소 접속
2. 사용자 소유 CSV 파일 업로드
3. 최적화 버튼 클릭

\# 탄소배출 Framework 설명 
1. 최적화 대상
2. 조직경계

\# 데이터 부분
1. 데이터 전처리 기능
2. 데이터 전처리 과정에서 발생하는 결과 사용자에게 전달 기능 (단위 변환/missing data,...)

\# 결과 부분
1. 사용자 결과 옵션 선택
2. 필요한 옵션 코드 추가
3. 결과 report 다운 기능 추가
